Steps: 1. Check the media queries in the `sizes` property. 2. Find the first matching value. For example, on a mobile device with a CSS pixel width of 390px and DPR = 3 (iPhone 13), given setting sizes="(min-width: 768px) 50vw, 100vw", it matches the `100vw` option. 3. Calculate the optimal image size: 390 × 3 × 100% (100vw) = 1170. 4. Find the corresponding match in the `srcset`.
